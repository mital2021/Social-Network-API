# Social-Network-API

## Description

It's an application that builds an API for a social network web application where users can share their thoughts, react to friends thoughts, and create a friend list.



## Built with 
    *   Express
    *   Mongoose
    *   MongoDB
    *   NoSQL
    *   Node.js
    *   Insomnia
