# Social-Network-API

## Description

It's an application that generates Team Profile by title from command line. This application runs with Node.js

## Installation
To run this application you need to install npm inquirer and Jest

## Built with 
    *   Express
    *   Mongoose
    *   MongoDB
    *   NoSQL
    *   Node.js
    *   Insomnia
